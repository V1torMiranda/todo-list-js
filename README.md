# Todo List - JavaScript

Projeto desenvolvido durante estudos em desenvolvimento front-end, com base em aulas práticas.

Foram realizadas adaptações visuais e pequenos ajustes no código para reforçar o aprendizado.

## Funcionalidades
- Adicionar tarefas
- Editar tarefas
- Remover tarefas
- Marcar como concluída
- Persistência com localStorage

## Tecnologias
- HTML5
- CSS3
- JavaScript
